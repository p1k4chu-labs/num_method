What is the Gauss-Seidel Method?

The Gauss-Seidel method is an iterative technique used to solve a system of linear equations, typically represented in the form Ax = b. It is an improvement over the Jacobi method because it uses the most recently updated values of x within the same iteration, which generally leads to faster convergence.

Core Concept

The method systematically iterates through each equation, solving for each variable xi while assuming the other entries of x are known. The key is that as soon as a new value for an xi is found, it is immediately used in the subsequent equations of the same iteration.

Component-wise Formula

The formula for each component xi at iteration (k+1) is:

$$x_i^{(k+1)} = \frac{1}{a_{ii}} \left( b_i - \sum_{j<i} a_{ij}x_j^{(k+1)} - \sum_{j>i} a_{ij}x_j^{(k)} \right)$$

Matrix Formula

The matrix form of the Gauss-Seidel method is derived by decomposing the matrix A into its diagonal (D), strictly lower (L), and strictly upper (U) triangular components: A = D + L + U.

The iterative formula can then be expressed as:

$$(D+L)x^{(k+1)} = b - Ux^{(k)}$$

Solving for x(k+1), we get the iteration matrix form:

$$x^{(k+1)} = -(D+L)^{-1}Ux^{(k)} + (D+L)^{-1}b$$

This is in the form x(k+1) = Hx(k) + C, where H is the iteration matrix and C is a constant vector.

Condition for Convergence

A sufficient condition for the method to converge is that the matrix A is strictly or irreducibly diagonally dominant. While other matrices may also converge, diagonal dominance is a common guarantee.
